# cdicts


## GM89: Greenberg and Møller (1989)

### System

Ca - K - Na - Cl - SO<sub>4</sub> - H<sub>2</sub>O

### Validity range

  * **Temperature:** 0 °C to 250 °C


## M88: Møller (1988)

### System

Ca - Na - Cl - SO<sub>4</sub> - H<sub>2</sub>O

### Validity range

  * **Temperature:** 25 °C to 250 °C

  * **Ionic strength:** 0 to ~18 mol·kg<sup>−1</sup>
